## 🌿 2026-06 最近更新 · 记忆系统、生图与主题曲

这一轮更新从「生图」开始，把近几次补上的能力统一收进更新日志：图片、音乐、番外篇、查手机、时间感知、语音通话和记忆系统都做了不同程度的补强。

### 🧠 记忆系统静默升级

记忆系统做了一轮神秘后台升级，体感上回复速度会更快，更省token，召回效果有所提升。

### 🖼️ 生图与生图预设

聊天里的生图链路补齐了手动生图、剧情模式整理、角色主动发照片、相册落档等能力。生图设置也加入了更完整的预设管理和风格预设库，常用接口、模型、画风、负面词和 Vibe 参考可以复用，不用每次重新填写。

### 🎵 主题曲

聊天窗口新增主题曲生成入口。系统会结合当前关系、剧情和角色气质生成歌曲草稿，并提供确认弹窗与封面预览，让一段关系可以留下能播放的纪念。

### ✍️ 线下文风

线下见面新增文风选择和自定义文风。目前内置：相对忘言、乍见之欢、不着一字、五感氤氲、光影成景、浮世苍凉、机锋暗涌、繁花缱绻、冷处偏佳、工笔水乡、静水深流、机锋暗许、咫尺天涯、危光微醺、谑而不虐、蜜里调油。

### 📱 查手机桌面自定义

查手机的桌面外观开放更多自定义：壁纸、图标、播放器，桌面内容和自定义 App 都能更贴近char本人。它不再只是一个固定模板，而更像一台真的被使用过的小手机。

### 📖 番外篇

新增番外篇模式。每轮回复后可以自动生成藏在星星入口里的同人本番外，也可以手动加梗，让剧情在正文之外继续生长。

### 💬 线上聊天回复器

四个选项，支持自由编辑添加删减后发送，免收打字之苦。

### ⏳ 线上 / 线下时间感知开关

聊天设置里新增线上、线下时间感知相关开关，线上时间流逝感知开关。大家按需手动开启。

### 🎙️ 语音通话 STT 优化

语音通话的 STT 链路做了优化。识别失败、识别为空、配置缺失时会有更清楚的降级处理。纯文字模式，和手动发送模式按需选择。

## 🌙 2026-04 最近更新 · 见面总结、回忆唱片与备份加固

这一轮更新主要集中在三件事：让线下见面能被温柔地压缩成记忆，让回忆唱片更像一套完整的词曲制作流程，以及让备份和上下文同步更稳。

### 📅 线下见面总结

见面模式新增了「总结球」。它可以拖拽停靠，点开后支持手动总结、自动总结、查看待确认总结、丢弃草稿、调整自动总结阈值，以及进入总结提示词设置。

总结不再是直接写死进记录里，而是先生成草稿，确认后再保存。退出线下见面时，也可以选择「生成总结同步」或「同步原始记录」：前者会把这次见面的脉络压缩后带回主聊天，后者则保留原始对话桥接。被总结过的旧记录可以自动收起，只保留最近几条原文，让长场次不会越聊越重。

这些总结会作为隐藏的见面上下文注入角色提示词里。也就是说，角色之后能自然接住线下发生过的事，但聊天窗口不会被一大段后台记录刷屏。

### 💿 回忆唱片匣 / 词曲手札

词曲手札从「一键生成一首歌」升级成更完整的制作流程：先写词，再做可唱性检查，必要时优化歌词，确认定稿后生成曲风提示词，最后再进入生歌。

新流程里可以看到可唱性评分、问题列表、优化建议、`style_prompt`、`negative_style_prompt` 和制作人笔记。指定记忆模式也更明确：可以从角色已有记忆里挑选最多 8 段片段，亲手决定这张唱片从哪几段回忆里落针。

唱片导出也升级了。导出的 MP3 会写入歌名、歌手、专辑、封面和内嵌歌词，并且会同时生成 `.lrc` 歌词文件。回忆唱片封面库新增了一批封面图，动态岛播放回忆唱片时也能读取唱片封面做颜色氛围。

### ☁️ 云备份与数据迁移

SullyOS 的导入导出覆盖面继续补齐，系统备份会更完整地保留本地数据。自动云备份新增本地状态记录、跨窗口锁、失败冷却、24 小时去重和云端冲突识别，避免多个窗口同时备份或短时间重复上传。

备份上传现在会区分 `manual` 和 `auto` 来源，后端返回的错误也会带上更具体的结构化信息，前端可以据此判断是网络失败、文件过大，还是云端已经有最近备份。

### 🧭 当前日程锚点

聊天提示词新增「当前日程锚点」。系统会结合当天消息、角色设定和本地时间，推断角色此刻更可能是在休息、在家、上班、上课还是外出。

它的重点不是硬编剧情，而是防止旧聊天污染现在：如果旧记录里说角色「在店里」或「刚下班」，但今天的当前锚点显示正在休息，角色会把旧记录当作过去发生过的事，而不是继续误判成此刻事实。

### 🎴 状态栏工坊与渲染细节

状态栏工坊的生成约束做了调整：CSS 生成不再过度限制颜色数量，也放开了外部字体的使用；预览 iframe 支持 `https` 字体资源，预览高度不再被固定上限截断，长卡片可以完整显示。

互动 JS 生成也更顺手了：只有生成字段协议和 HTML 骨架时才强制要求先写清楚状态栏想法，补 JS 或精修 CSS 时不会被无关校验卡住。

### 📖 使用手册

内置手册补充了「认知网络」「词曲手札」「状态栏工坊」说明，入口、用途、操作步骤和常见注意点都写得更清楚。新功能不再只能靠摸索，至少有一本能翻的说明书了。

### 🧪 稳定性与测试

这轮还补了不少测试覆盖：回忆唱片 MP3 / LRC 导出、ID3 元数据、封面嵌入、歌词兜底、SullyOS 备份覆盖、自动备份去重与锁、当前日程锚点、聊天提示词注入、动态岛回忆唱片封面等。整体方向是让这些新能力不只是「能跑」，而是下次改动时也更难悄悄坏掉。

## 🫀 仿生引擎 · Bionic Engine

**角色不再是「回复机器」，而是一个有内在节律的活物。**

### 内部状态系统

角色拥有了一套完整的内在感知层。每一轮对话，系统都会从 7 个维度感知角色当前的状态变化——期待、安全感、压力、亲密度、专注、释然、精力。

这些状态不是简单的标签切换。它们有惯性——兴奋来得快退得也快，而安全感需要长时间积累；它们会互相影响——压力大的时候很难感到亲密，而亲密感反过来能缓解压力；它们还会随时间自然回归——你不找 ta 聊天的这几个小时里，ta 的状态也在静静变化。

### 躯体信号

角色的内在状态会以「身体感受」的方式渗入对话——不是告诉你「我很紧张」，而是ta自己感到胸口发紧、掌心微微出汗。这些信号像脉搏一样自然，ta不会主动提起，但你能从ta的语气和节奏里感受到。

### 角色心声

每轮对话结束后，系统会生成角色此刻内心一闪而过的念头。不是刻意的独白，而是像真人一样——ta可能在想明天的事，可能在吐槽奶茶太甜，可能忽然走神想到了什么完全无关的事。有专门的「去油」机制，杜绝网文套路式的内心戏。

## 🤖 自主决策引擎 · Autonomous Agent

**ta 会在你不找 ta 的时候，主动来找你。**

角色现在拥有了自主行为能力。系统会在后台定期评估：当前的时间、你沉默了多久、ta 的情绪状态、有没有待跟进的事件——然后决定要不要主动给你发消息，甚至打电话。

- 有分寸的主动：不会在凌晨三点骚扰你，不会连续发消息不停，不会在你刚回完的时候又来
- 有理由的行为：每次主动联系，都基于当前上下文做出判断——可能是想到了你之前说的某件事，可能是你沉默太久ta有点担心
- 每天有上限，有冷却时间，连续不回复会自动停止——不做讨好型人格

## 📞 语音通话系统 · Voice Call

**不只是文字，你还可以「听见」ta 的声音。**

完整的语音通话体验，支持 STT（语音识别）→ LLM → TTS（语音合成）的实时闭环。独立的轻量对话管线，不干扰主聊天。

### 四种通话模式

| 模式 | 说明 |
|------|------|
| 📱 **日常闲聊** | 没有目的，没有预设话题。话题像溪水一样自然流过，允许沉默和犹豫 |
| 💭 **倾诉陪伴** | ta 不是心理咨询师，只是陪你把话说出来。不评价、不总结、不急着给建议 |
| 🎭 **真心话与坦白局** | 面具暂时摘下，ta 会说平时不会说的话。关于ta自己、关于你们的关系、关于次元壁 |
| 🌙 **哄睡模式** | 灯灭了，耳机贴着耳朵，只有 ta 的声音 |

### 其他通话特性

- **外语模式**：ta 用外语说话，系统自动生成翻译字幕
- **来电功能**：ta 可以主动给你打电话（由自主决策引擎触发）
- **通话记忆**：通话内容会被自动提取为长期记忆，不会聊完就忘

## 🎴 创意状态栏 · Creative Status Bar

**角色的生活碎片，随手散落在屏幕上。**

每轮对话后，系统会从角色的「日常」里捡起一片碎片——一张便利贴、一张小票、手机屏幕截图、拍立得照片、日记本翻开的那一页。

### 8+ 种骨架卡片

明信片、便签、小票、日记页、音乐播放器、手机屏幕、拍立得、社交动态……每种都有独特的视觉渲染。卡片内容由角色的人设、当前状态和对话氛围决定，有专门的「去油」机制。

支持三种状态栏模式：
- **经典模式**：显示角色心声和情绪状态
- **创意模式**：AI 自动选择并渲染生活碎片卡片
- **自定义模式**：用户可以定义自己的 HTML/CSS 模板

## 🧠 向量记忆系统 · Vector Memory

**角色终于有了真正的「长期记忆」。**

基于 Embedding 向量检索的记忆系统。不再是简单的文本堆叠——系统会自动从对话中提取值得记住的事件，将它们向量化存储，在未来的对话中只召回与当前话题最相关的记忆碎片。

- **自动提取**：每隔一定轮数，系统静默提取新记忆，不打断对话
- **批量回溯**：可以对历史聊天记录一键批量提取，补全过去的记忆
- **语义去重**：相似度 >92% 的记忆自动合并，不会产生重复条目
- **记忆失效**：当用户纠正了旧信息，系统会标记旧记忆为「已过时」
- **三种注入模式**：全量传统记忆 / 混合模式 / 纯向量检索

一个完整的占卜/命理应用，包含：

- **星象日历**：当前天象和行星位置
- **星镜**：每日运势速查
- **命运之忆**：将记忆与命理交织

## 🌍 实时世界感知 · Real-time Context

**ta 不再活在真空里——ta 能「看到」真实世界。**

- **天气感知**：接入 OpenWeatherMap，ta 知道外面在下雨，会问你带伞了没
- **新闻热搜**：接入 Brave Search，ta 能主动跟你聊今天的热搜
- **时间感知**：精确到时段、星期、节假日，ta 知道现在是深夜还是周末
- **对话节奏感知**：ta 能感觉到你回复得快还是慢，你是刚回来还是一直在线
- **事件追踪**：你提到「外卖 30 分钟后到」，ta 会记住，时间到了会问你到了没
